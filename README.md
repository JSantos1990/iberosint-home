# IberOSINT

Official homepage of the IberOSINT platform.

## Description

IberOSINT is an OSINT platform developed as a Master's Final Project focused on providing investigators and cybersecurity professionals with a centralized collection of Open Source Intelligence resources.

The homepage provides quick access to infrastructure search engines, malware analysis platforms, social media intelligence, DNS investigation tools, geolocation resources, Onion services and many other OSINT utilities.

## Features

- Modern interface
- Dark theme
- Fast search
- Organized by categories
- Compatible with Tor Browser
- Single HTML file
- Static website
- Hosted with GitHub Pages

## Categories

- Infrastructure OSINT
- DNS Intelligence
- Malware Analysis
- Credential Leaks
- SOCMINT
- Geolocation
- Onion Resources
- Correlation Tools
